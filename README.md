🔐 Brute Force Password Attack Simulation (Python)

This project demonstrates how brute-force attacks work by randomly guessing
a password using a predefined character set.  
It is created for **educational purposes only** to understand password security.

🚀 Features
- Random password generation
- Attempt counter
- Time taken calculation
- Console-based simulation

🛠 Technologies Used
- Python
- random
- time
- os

📚 What I Learned
- Brute-force attack concepts
- Time complexity of password cracking
- Importance of strong passwords
- Python loops and randomness

⚠ Disclaimer
This project is for learning and demonstration purposes only.
It should NOT be used for illegal or unethical activities.
